## Features

- **Beautiful Modern UI:** Comes with nice animations and a familiar yet sleek GUI.
- **YT Music Support:** All of your library, subscriptions, history is in one place.
- **Customiseable:** Apply different themes, or create your own.
- **Fast:** Everything gets fetched blazingly fast.
- **Discord/Last.fm Integration:** Show what you're listening to in both platforms.
- **Real-Time Lyrics:** Watch the lyrics go by as you listen.
- **Not Electron:** Yeah, thats a feature...
- **And more!**

## Getting Started

### Prerequisites
- Visual Studio 2022 (or your preferred C# IDE)
- .NET Desktop Development Workload

### Build & Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Alpointernet/Spectre.git
   ```
2. Navigate to the project directory and open `Spectre.csproj` in Visual Studio.
3. Restore any required NuGet packages.
4. Build the solution and press `F5` to run the application.

*Alternatively, you can run the `Build.bat` script to compile the project via the command line.*
